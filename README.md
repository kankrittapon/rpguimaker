# RPGUIMaker

**No-Code UI Engine for RPG and Custom GUIs in Minecraft 1.21.1**

RPGUIMaker is a NeoForge mod designed to revolutionize how you create custom User Interfaces in Minecraft. It features a powerful engine that allows for complex UI designs without writing code.

## ‚ú® Features

- **No-Code UI Design**: Create stunning RPG-style interfaces using configuration and layout files.
- **Advanced Rendering**: Support for complex visual elements and styles.
- **Alpha Masking**: Create non-rectangular, shaped UI elements with texture masking.
- **Point-in-Polygon Interaction**: Precise click detection for irregular shapes, ensuring buttons and interactive elements feel natural.
- **Dynamic Textures**: Load and manage textures at runtime.
- **NeoForge 1.21.1 Support**: Built for the latest modern modding standards.

**Windows:**


```bash

```

The compiled jar file will be located in `build/libs/`.
