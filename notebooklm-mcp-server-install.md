ติดตั้งและตั้งค่า NotebookLM MCP Server
1.ติดตั้งแพ็กเกจ notebooklm-mcp-server ด้วย uv (เช็คก่อนว่ามี uv ติดตั้งไหม)
2.เพิ่มการตั้งค่าในไฟล์ MCP configuration ของผม (C:\Users\zexqm\.gemini\antigravity\mcp_config.json)
3.ยืนยันตัวตนด้วย uvx --from notebooklm-mcp-server notebooklm-mcp-auth
4.ตรวจสอบว่าติดตั้งเสร็จแล้วและใช้งานได้จริง
\*\*Browser เป็น edge ที่ผมเปิดอยู่ในขณะนี้ได้ยิ่งดี
